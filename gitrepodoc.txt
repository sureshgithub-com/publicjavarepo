1. clone the git repository to the local location
>> $ git clone https://github.com/sureshgithub-com/publicjavarepo.git

2. Change directory to publicjavarepo folder
>> $ cd publicjavarepo/

3. Create a new branch 
>> $ git checkout -b reshmaraju origin/release/1.0.0
branch 'reshmaraju' set up to track 'origin/release/1.0.0'.
Switched to a new branch 'reshmaraju'

4. Create any java program
>> $ vi firstprogram.java

5. Check the git status 
>> $ git status
On branch reshmaraju
Your branch is up to date with 'origin/release/1.0.0'.

Changes not staged for commit:
  (use "git add <file>..." to update what will be committed)
  (use "git restore <file>..." to discard changes in working directory)
        modified:   firstprogram.java

6. Add the newly created file to the stash
>> git add firstprogram.java 

7. commit the file to the repository 
>> $ git commit -a -m "Reshma - my first program"
